# CyberDrops cyber avatars

Original fictional avatar collection generated for CyberDrops with OpenAI image
generation. The collection uses a cohesive premium 3D cyberpunk style with
robots, gamer mascots, hardware, neon, RGB lighting and glitch effects.

The source artwork was generated as a consistent 4x3 sprite sheet and cropped
into twelve square PNG assets:

- NeonHunter
- PixelStorm
- CyberLoot
- RTXQueen
- GameDropper
- BitNinja
- LootMaster
- GlitchCore
- RGBPilot
- PCTitan
- ChipSpark
- HeadsetZero

`cyber-fallback.png` is the local fallback used whenever an avatar cannot load.
No photograph or real human face is used in this collection.
